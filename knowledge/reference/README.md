# CL Reference Cache

Cached documentation snippets to reduce web lookups.

## Key Resources
- **CLHS (HyperSpec):** http://www.lispworks.com/documentation/HyperSpec/Front/
- **CL Cookbook:** https://lispcookbook.github.io/cl-cookbook/
- **SBCL Manual:** https://www.sbcl.org/manual/
- **Quicklisp:** https://www.quicklisp.org/
- **Alexandria:** common utilities library
- **Dexador:** HTTP client
- **Jonathan/Jzon:** JSON libraries
- **Bordeaux-threads:** portable threading
- **Usocket:** portable sockets

## Usage
Cache frequently-needed doc snippets here as .md files to avoid repeated web fetches.
When looking something up, save the relevant excerpt for next time.
